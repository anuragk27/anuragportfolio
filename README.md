# My Portfolio

A modern and responsive **portfolio website built using Flutter**. This project showcases my work, skills, and personal information with a clean UI and smooth user experience, optimized specifically for web.

---

## 🚀 Features

*  Clean and minimal UI
*  Projects showcase
*  About me section
*  Skills & tools
*  Contact section

---

## 🛠 Tech Stack

* **Framework:** Flutter (Web)
* **Language:** Dart
* **UI:** Material Design / Custom Widgets


## 📦 Installation & Run (Web)

1. Clone the repository:

```bash
git clone https://github.com/your-username/flutter-web-portfolio.git
```

2. Navigate to the project folder:

```bash
cd flutter-web-portfolio
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run in browser:

```bash
flutter run -d chrome
